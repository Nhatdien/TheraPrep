
### Goal:

Allow users to find lessons by keyword or topic using existing `collections` and `slide_groups`.

### 🔧 Solution:

Use **PostgreSQL full-text search** + **vector search** (Qdrant) for hybrid retrieval.

#### API Endpoint (Go)

`GET /api/lessons/search?q=mindfulness`

#### Backend flow:

1. **Primary search** → use PostgreSQL full-text search on:
    
    - `collections.title`
        
    - `collections.description`
        
    - `slide_groups.content::text`
        
2. **If no strong matches** → send query to Python AI service:
    
    - Use Qdrant embeddings for semantic retrieval (based on HuggingFace model).
        
    - Return related lessons with confidence score.
