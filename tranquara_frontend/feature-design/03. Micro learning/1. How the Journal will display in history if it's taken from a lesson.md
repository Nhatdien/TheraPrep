
### Goal:

Integrate journals written inside lesson slides (e.g., `journal_prompt` slide) into the **Journal History** page — while still linking them to their source lesson.

### 🔧 Solution:

Add two new optional fields in your `user_journals` table:

|Column|Type|Description|
|---|---|---|
|`collection_id`|UUID (nullable)|FK → `collections.id`, set when journal comes from a lesson|
|`source_type`|VARCHAR(50)|`"lesson"` or `"standalone"`|

#### Example Entry

|id|title|content|collection_id|source_type|
|---|---|---|---|---|
|a1b2|“Noticing Anger”|“I realized anger…”|`c7a6d9...`|`"lesson"`|

#### Journal Page UI logic:

- If `source_type === "lesson"` → show a **link or badge**:
    
    > 🧠 _From lesson: “Naming Emotions Reduces Their Power”_  
    > → tapping it reopens the original lesson.
    

This keeps journaling consistent and helps users see where reflections came from.
